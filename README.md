# CocktailsCategories
**CocktailsCategories based on Free Cocktail API**
<br />
**It make retrieving remote cocktails and filtering them**
<br />
* Architecture - **MVVM**
* Layout - **Snapkit**
* Bindigs - **Closure**
* Networking - **Moya**
* Resources - **SDWebImage**
